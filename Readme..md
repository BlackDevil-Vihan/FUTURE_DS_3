# Marketing Funnel & Conversion Performance Analysis Dashboard

## Overview

This project presents a **Marketing Funnel & Conversion Performance Analysis Dashboard** developed using **Power BI**. The dashboard analyzes customer marketing campaign data to identify conversion patterns, customer behavior, campaign effectiveness, and areas of improvement in the lead-to-customer conversion process.

The objective of this project is to transform raw marketing data into meaningful business insights through interactive visualizations and key performance indicators (KPIs).

## Key Features

* Total Leads Analysis
* Converted Customer Tracking
* Conversion Rate Monitoring
* Campaign Contact Analysis
* Customer Segmentation by Job, Marital Status, and Housing Loan Status
* Non-Converted Customer Analysis
* Monthly Conversion Trend Analysis
* Interactive Filtering and Drill-Down Capabilities

## Dashboard KPIs

* Total Leads
* Converted Customers
* Conversion Rate (%)
* Average Call Duration
* Total Campaign Contacts
* Customer Drop-Off Analysis

## Visualizations Used

* KPI Cards
* Bar Charts
* Pie Charts
* Donut Charts
* Treemap
* Gauge Chart
* Line and Stacked Column Chart
* Interactive Slicers

## Business Insights

* Identifies customer segments with the highest conversion rates.
* Highlights job categories contributing most to successful conversions.
* Analyzes the impact of housing loans and marital status on customer subscriptions.
* Tracks non-converted customers to uncover potential improvement opportunities.
* Monitors campaign performance and customer engagement trends over time.

## Tools & Technologies

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Data Visualization Techniques
* Marketing Analytics

## Outcome

The dashboard provides a comprehensive view of marketing campaign performance, enabling data-driven decision-making and helping businesses optimize customer acquisition and conversion strategies.
